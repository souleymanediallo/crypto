Projet DJANGO

Le site crytocompare.com propose des données détaillées sur diverses crypto-monnaies, y compris des informations sur les prix, le volume des échanges, et les mouvements des marchés. L'utilisation de l'API est gratuit pour les besoins personnels. Ce projet illustre l'utilisation d'une API externe pour enrichir un site web avec des données dynamiques, tout en utilisant le puissant framework Django pour gérer le backend.

cryptocompare.com

Base de données

Bien que le projet puisse fonctionner sans base de données locale (en récupérant directement les données de l'API), je pourrais également intégrer une base de données pour stocker des informations sur les utilisateurs, leurs préférences, ou pour mettre en cache certains articles.
